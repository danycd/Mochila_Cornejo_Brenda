Comandos mas utilizado en Git-GitHub
git init: este comando permite iniciar un repositorio nuevo dentro de nuestro computador es decir un nuevo repositorio local.
git remote add origin <link de repositorio remoto>: este comando permite conectar nuestro repositorio local con nuestro repositorio Remoto en GitHub.
git add . : agrega todos los archivos y los deja en estado tracked y listos para realizar un commit
git commit -m "mensaje descriptivo del commit": crea un punto donde todos los datos añadidos con git add se suben a nuestro repositorio local
git push origin main: sube a nuestro repositorio remoto todos los commits realizados en nuestro repositorio local
git clone <link de repositorio remoto>: crea un repositorio local haciendo una copia del repositorio remoto que se coloque en el link
git pull origin main: actualiza nuestro repositorio local con todos los cambios que haya sufrido el repositorio remoto
git checkout <nombre de la rama>: cambia a la rama especificada
git checkout <hash>: coloca al repositorio en el estado en el que se encontraba en el commit especificado con su hash
git branch <nombre rama>: crea una rama con un nombre especificado
git status: muestra el estado de los archivos
git config user.name=<nombre de usuario>: configura el usuario con la especificación dada
git config user.email=<email del usuario>: configura el email con la especificación dada
